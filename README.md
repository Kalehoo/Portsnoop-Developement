
# Kalevi's Portsnoop (Developement version)
 Tool to sniff open ports from system

 You need latest Golang version to run this project.

Usage:
```
  -h string
        Defines host
  -help
        Shows this help
  -p int
        Defines port for lookup
  -p* string
        Defines series of ports
  -pr string
        Defines range of ports.
```
